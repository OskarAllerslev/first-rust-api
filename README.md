
# Rust-api 

I want to try and create a rust api. 

Mostly for learning. 

Initially i want to just set it up and see how it works. 

Then i would like to be able to fetch stock data with it. 

Perhaps later on fit an xgb. 


# How to run

You should run **cargo run** and then visit the localhost "" for the api

# Endpoints

- alive_check: can be run with "http://localhost:3000/is_alive"

- get_ticker_data: can we run with for example: "http://localhost:3000/data/AAPL?interval=1d&range=1d"
